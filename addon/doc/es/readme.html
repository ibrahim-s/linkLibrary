Link Library

Autor: Ibrahim Hamadeh
Descargar Versión 2.2.6
Compatibilidad con NVDA: 2021.1 en adelante

Este complemento ayuda al usuario a organizar sus enlaces o marcadores al estilo de una biblioteca.
Desde el diálogo principal de bibliotecas, puedes añadir, renombrar o eliminar cualquiera de ellas.
Si ya tienes bibliotecas, simplemente elige la categoría que desees y pulsa Intro sobre ella.
En el diálogo que se abre, verás todos los enlaces de esa categoría en una lista y tendrás acceso a la URL de origen y a la información "Acerca de" de cada enlace, si está presente.
Este complemento no viene con un gesto o atajo de teclado asignado por defecto.
Como siempre, puedes añadir un gesto o cambiar el existente yendo a:
Menú NVDA > Preferencias > Gestos de entrada > Link Library.

Uso

  • Tras asignar un gesto al complemento, utiliza ese gesto o atajo para abrir el diálogo principal.
  • Se abrirá un diálogo listando todas las bibliotecas o categorías de enlaces encontradas.
  • Al principio no se encontrará ninguna biblioteca; estará vacío, listo para alojar las bibliotecas que desees crear.
  • Tabulando en ese diálogo, puedes renombrar, eliminar o añadir cualquier biblioteca o categoría de enlaces en cualquier momento.
  • Si ya tienes bibliotecas, sitúate sobre la de tu elección en la lista y pulsa Intro.
  • Se abrirá un diálogo para esa biblioteca, mostrando todos los enlaces (nombre o etiqueta) en una lista.
  • Al principio, la biblioteca estará vacía y no habrá enlaces.
  • Pulsamos el menú contextual (Shift + F10), o la tecla de aplicaciones, para añadir un enlace introduciendo la URL, la etiqueta y, opcionalmente, la información "Acerca de" del enlace.
  • Al situarse sobre cualquier enlace de esta lista, pulsar Intro lo abrirá con el navegador predeterminado.
  • Tabulando en ese diálogo puedes acceder a la información del enlace (si existe), a un botón para mostrar la URL de origen y a otro botón que te permite abrir el enlace con diversos navegadores instalados en tu equipo.
  • Utilizando el control de filtrado, puedes filtrar los enlaces que contengan palabras o frases específicas en la biblioteca.
  • Sitúate sobre el enlace que desees y pulsa el menú contextual.
  • Desde allí puedes añadir, editar, mover el enlace a otra biblioteca, o eliminar ese enlace o todos.
  • Si deseas importar una biblioteca JSON o exportar cualquier biblioteca como archivo JSON o HTML, puedes hacerlo; además, puedes importar todos los marcadores de Firefox como JSON.
  • Para exportar una biblioteca, sitúate sobre ella y pulsa el menú contextual, baja con la flecha hasta "Exportar biblioteca como", elige el tipo de archivo (JSON o HTML) y pulsa Intro.
  • Se abrirá un diálogo para elegir la carpeta de destino; elígela y pulsa Intro o navega hasta el botón "Seleccionar carpeta".
  • ¡Listo! Aparecerá un mensaje informativo indicando que la biblioteca ha sido exportada.
  • Un detalle interesante: si exportas una biblioteca como HTML, puedes incluir esos enlaces en los marcadores de Firefox.
  • Solo abre Firefox y, desde el menú de marcadores, elige "Mostrar todos los marcadores" (Control + Shift + B).
  • En el menú de la nueva ventana, ve al submenú "Importar y respaldar", luego a "Importar marcadores desde HTML..." y pulsa Intro.
  • Navega por el diálogo hasta el archivo HTML y selecciónalo. Se creará una carpeta con el mismo nombre de la biblioteca importada en el menú de marcadores de Firefox.
  • Por último, pero no menos importante, tienes la capacidad de elegir la ubicación de los archivos de datos del complemento.
  • Al instalar el complemento por primera vez, aparecerá un mensaje preguntando si aceptas crear la carpeta que aloja los datos en tu directorio de usuario.
  • Si lo deseas, puedes pulsar "Cancelar" y elegir el directorio más tarde desde el diálogo de configuración del complemento en el menú de preferencias.
  • Ve a Menú NVDA > Preferencias > Link Library > Link Library Setting... y abre el diálogo de ajustes.
  • Mediante el botón "Añadir nueva ruta", puedes añadir una o más rutas, o eliminarlas con el botón "Eliminar".
  • Tras elegir la nueva ruta predeterminada para los archivos de datos, si quieres que sea permanente, debes guardar la configuración pulsando Control + NVDA + C (especialmente si no tienes activado "Guardar configuración al salir").
  • Cabe destacar que crear la carpeta de datos en el directorio de usuario permite que todas las instancias del complemento compartan la misma base de datos.
  • Elegir una ruta como Dropbox, por ejemplo, permitirá compartir la misma base de datos con instancias del complemento en otros ordenadores que usen la misma cuenta de Dropbox.

Función de Sub-bibliotecas

Puedes añadir sub-bibliotecas a cualquier biblioteca principal.

  • Abre la biblioteca que desees.
  • Pulsa la tecla de menú contextual o aplicaciones.
  • Desde allí tendrás la opción de añadir, renombrar o eliminar cualquier sub-biblioteca presente. ¡Fácil y útil!

Añadir un enlace al vuelo

Una función fantástica: añadir el enlace de una página web sobre la marcha.

Para usar esta función, primero debes asignar un gesto en: Menú NVDA (NVDA+N) / Preferencias / Gestos de entrada / Link Library / Añadir el enlace y título de la página web al vuelo a la biblioteca que elijas.

Ahora, mientras navegas, si quieres añadir una página a cualquier biblioteca, pulsa el gesto asignado desde cualquier parte de la página. Se abrirá una ventana titulada "Añadir enlace y título de página web a la biblioteca". Pulsa el botón "Elegir biblioteca:", elige la que desees en el menú emergente y pulsa Intro.

Eso es todo; el enlace se añadirá con el título de la página como etiqueta y escucharás un mensaje de confirmación.
